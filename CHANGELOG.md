# Changelog

## [0.1.1] - 2026-03-13

### Changed
- namespace of `ISpriteTKBundleLoader` has been changed to the `RisSpriteToolkit`.

## [0.1.0] - 2026-03-12

### Added

- `SpriteTKBundleBuilder` which can be used to build a sprite sheet bundle.
- `SpriteTKBundleLoader` which can be used to load a sprite sheets bundle.
